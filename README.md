# UWAFT Software Challenge
Welcome to the UWAFT "Hack the Move" 2026 hackathon! In this repository you will find all the necessary guides and files to complete the software challenge. This challenge will introduce you to the the controls and programming behind autonomous driving. You will be using MATLAB and Simulink to write your code and simulate it on a virtual enviornment called QLabs which will show how the car respobds to your programming.

## Introduction
In this Hackathon you will be playing the role of an autonmous taxi service.
